# TOEFL Prep & English Mastery Web App

A comprehensive, AI-powered platform designed to help students master English grammar and prepare for the TOEFL iBT and IELTS exams. This application provides a structured learning path from elementary foundations to advanced academic proficiency.

## 🚀 Key Features

### 1. Grammar Foundations
Structured lessons categorized by educational levels:
- **Elementary School:** Basic nouns, verbs, and adjectives.
- **Junior High School:** Tenses, conjunctions, and sentence structures.
- **High School:** Conditionals, passive voice, and complex academic structures.

### 2. Academic Vocabulary
- **Interactive Flashcards:** Master high-frequency academic words.
- **AI-Powered Generation:** Generate new vocabulary sets based on specific topics (Academic, Scientific, Business, etc.) using Gemini AI.
- **Contextual Learning:** Each word includes definitions, examples, and synonyms.

### 3. AI Speaking Practice
- **Real-time Feedback:** Practice speaking with AI-generated prompts.
- **Detailed Analysis:** Receive instant, professional feedback on pronunciation, fluency, and grammar.
- **Integrated Audio:** Record and review your responses directly in the browser.

### 4. Full Test Simulations
- **TOEFL iBT Simulation:** Academic-focused tasks mirroring the real exam structure.
- **IELTS Academic:** Comprehensive evaluation of Reading, Listening, Writing, and Speaking skills.
- **Proficiency Tracking:** Monitor your progress with an overall proficiency score.

### 5. Listening Comprehension
- **Engaging Content:** High-quality audio lessons focused on campus life and academic scenarios.
- **Interactive Quizzes:** Test your understanding with immediate feedback and explanations.

## 🛠️ Technical Stack

- **Frontend:** React 18 with Vite
- **Styling:** Tailwind CSS (Indigo & Amber Theme)
- **Animations:** Framer Motion
- **Icons:** Lucide React
- **AI Integration:** Google Gemini API (@google/genai)
- **State Management:** React Hooks

## 💻 Getting Started

### Prerequisites
- Node.js (v18 or higher)
- npm or yarn

### Installation

1. **Clone the repository:**
   ```bash
   git clone <repository-url>
   cd toefl-prep-app
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Environment Setup:**
   Create a `.env.local` file in the root directory and add your Gemini API key:
   ```env
   GEMINI_API_KEY=your_api_key_here
   ```

4. **Run the development server:**
   ```bash
   npm run dev
   ```
   The app will be available at `http://localhost:3000`.
